# DataMind AI

An intelligent AI-powered platform built with Next.js, React, Node.js, and Prisma.

## 🚀 Live Demo
Check out the live deployment here:  
👉 **[https://datamind-ai-ypn1.vercel.app/](https://datamind-ai-ypn1.vercel.app/)**

---

## 🛠️ Tech Stack
* **Frontend:** Next.js 16 (Turbopack), React, Tailwind CSS
* **Backend & Database:** Node.js, Prisma ORM, Relational Database
* **Deployment:** Vercel (Frontend & Edge Routing), GitHub CI/CD
